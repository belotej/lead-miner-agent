# Lead Miner Agent

This project aims to build a Lead Miner Agent consisting of three separate agents:

1. **Discovery Agent**: Responsible for finding potential leads.
2. **Enrichment Agent**: Responsible for gathering more details about the leads.
3. **Qualification Agent**: Responsible for qualifying the leads based on specific criteria.

## Project Structure

- `/docs`: Planning and documentation files.
- `/discovery-agent`: Code for the Discovery Agent.
- `/enrichment-agent`: Code for the Enrichment Agent.
- `/qualification-agent`: Code for the Qualification Agent.
